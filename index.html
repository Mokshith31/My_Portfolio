My Projects

Audio/Text to sign language

The project aims to develop a website that converts English text into Indian Sign Language (ISL) to facilitate communication between hearing-impaired individuals and others. The system uses audio input, converts it to text, and then uses natural language processing (NLP) to interpret and choose the appropriate ISL signs. The software is written in Python and uses libraries such as Django and NLP, and the hardware required includes a device with a microphone, monitor, and internet connectivity.
